Desenvolvimento Web

01

Criação de Ecossistemas Responsivos

Meus projetos são desenvolvidos com foco total em **performance** e **adaptabilidade**. Utilizo HTML5 semântico e CSS3 avançado (Flexbox e Grid) para garantir que a interface seja impecável em qualquer dispositivo.

Como foi desenvolvido: O foco inicial foi a estruturação de variáveis CSS para facilitar a manutenção e garantir uma identidade visual coesa em todas as páginas, como visto neste portfólio.

UI/UX Design

02

Interfaces Centradas no Usuário

Baseado na certificação do Google, aplico metodologias de design thinking e prototipagem para criar fluxos que convertem e encantam o usuário final.

Como foi desenvolvido: Através de pesquisas de usuário e criação de personas, definindo paletas de cores (como o azul #007BFF dominante) que transmitem confiança e profissionalismo.

Análise e Arquitetura de Software

Aplicação dos conceitos de lógica de programação e engenharia de software aprendidos na FATEC para criar códigos limpos, escaláveis e seguros.
